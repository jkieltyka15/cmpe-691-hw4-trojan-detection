Prerequisites: gcc with C11 compiler using Linux

Part B:
    1) To build, use command 'make parsedata' in the 'project root' directory.
    2) To perform parsing, use command './build/bin/parse-data ./data/input-data.txt ./data/ouput-data.txt' 
       in the 'project root' directory.
    3) Output will be in 'output-data.txt' in the 'data' directory.

Notes
    * Simply using the command 'make' will build everything.
    * To clean the 'build/bin' directory, use the command 'make clean'.
